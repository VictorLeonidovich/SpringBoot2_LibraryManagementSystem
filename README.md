# 📚 Library Management System

Веб-приложение для управления библиотечной системой, построенное на базе Java-фреймворка Spring Boot.

**GitHub Repository:** [VictorLeonidovich/LibraryManagementSystem](https://github.com/VictorLeonidovich/LibraryManagementSystem)

---

## 🛠 Технологический стек

* **Backend:** Java, Spring Boot 4, Spring Data JPA (Hibernate)
* **Frontend:** HTML5, Thymeleaf
* **Сборка и зависимости:** Maven, MapStruct
* **Базы данных и миграции:** Liquibase
* **Мониторинг и валидация:** Spring Boot Actuator, Jakarta Validation

---

## 🏛 Архитектура проекта

Приложение реализует классическую монолитную архитектуру:
* **UI-слой:** Серверный рендеринг (SSR) веб-страниц с помощью Thymeleaf.
* **API-слой:** Выделенный REST API с DTO-маппингом (MapStruct) и собственным обработчиком исключений.
* **Бизнес-логика:** Сервисный слой на Java для обработки правил библиотеки.
* **Данные:** Модель данных (JPA-сущности) для книг, авторов, категорий и издателей.
